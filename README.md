## 간단한 회고
</br>
  
- 보완점  
1. 서버의 예외처리가 덜 되어 잘못된 요청에 취약함 </br>
2. 기획했던 기능을 전부 구현하지 못 함 </br>
3. TodoList 반응형 미구현 </br>
4. TodoList 클릭시 아래로 내려가는 현상 미해결 </br>
5. 마우스 포인터를 세세하게 신경쓰지 못 함 </br>
6. 캘린더에 공휴일을 표시하지 못 함(구글 캘린더 API 사용하고자 했음) 
  
</br>
  
- 이번 프로젝트를 통해 얻은 점  
1. 프론트와 백엔드를 한 번에 아우르는 프로젝트를 진행하여 좋은 경험이 됨</br>
2. 잘 몰랐던 API나 암호화같은 부분을 공부할 수 있는 기회가 됨</br>
3. 실용적인 결과물을 무사히 배포한 값진 경험이 되었음</br>
